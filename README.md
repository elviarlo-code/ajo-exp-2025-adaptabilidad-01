# Experimento 2025 – Evaluación de seis cultivares de ajo en Lambayeque

Repositorio del ensayo experimental correspondiente al proyecto PN_Hortalizas_Ajo – campaña 2025.

Este repositorio contiene los datos, protocolos, análisis y borradores del artículo científico asociados a la evaluación inicial de adaptabilidad de seis cultivares de ajo en la región Lambayeque.

---

## 🎯 Objetivo del experimento

Evaluar el comportamiento agronómico, rendimiento y adaptación inicial de seis cultivares de ajo bajo condiciones agroclimáticas de Lambayeque, como base para el desarrollo de una tecnología regional de semilla.

---

## 🧪 Material vegetal

- 3 cultivares comerciales de ajo  
- 3 colecciones de ajo  

(Total: 6 cultivares)

## 📊 Variables principales

### Variables de planta
- Tallo flora curvatura 
- Tallo floral bulbillos
- Follaje - porte
- Número de hojas
- Forma longitudinal
- Posición de dientes-extremo de bulbo
- Forma de la base
- Dsictibución de dientes
- Dientes experiores
- Días a emergencia
- Días a formación de bulbo

### Variables de rendimiento
- Peso promedio de bulbo
- Peso total por parcela
- Rendimiento estimado (t/ha)
- Número de dientes
  
### Variables de calidad
- Longitud de bulbo
- Ancho de bulbo
- Uniformidad
- Clasificación comercial
